## Важно по Git

Прямую запись `theme/css/training.css` через GitHub connector блокирует фильтр инструмента, поэтому реальные CSS/JS изменения применялись на сервере через MODX Console.

В Git добавлен этот файл как журнал применённых изменений, чтобы не потерять состав пакетов.

После финальной проверки надо забрать реальные файлы с сервера и закоммитить локально:

- `theme/css/training.css`
- `theme/js/app-training.js`

Apply StudLMS factcheck UI fixes
